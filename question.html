Question-1: What are the different ways ot select an element in the DOM?

In JavaScript, there are several ways to select elements in the DOM (Document Object Model). 1.getElementById() – select an element by id. 2.getElementsByName() – select elements by name. 3.getElementsByTagName() – select elements by a tag name. 4.getElementsByClassName() – select elements by one or more class names. 5.querySelector() – select elements by CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent?

The innerHTML, innerText, and textContent properties in JavaScript are used to access or modify the content of an HTML element. 1. innerHTML:innerHTML returns or sets the HTML content inside an element, including all HTML tags and text. 2. innerText:innerText returns or sets the visible text content of an element, excluding any HTML tags. 3. textContent: textContent returns or sets the text content of an element, including all child nodes, but excludes HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where I can attach a single event listener to a parent element, rather than attaching individual event listeners to each of its child elements. This allows the parent element to handle events that occur on its descendants

Question-4: What is event bubbling in the DOM?

In the Document Object Model (DOM), "event bubbling" refers to a phase in the event propagation process where an event, once triggered on a specific element, "bubbles up" through its ancestor elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript provides powerful tools for dynamically manipulating the DOM (Document Object Model), allowing us to create, add, and remove HTML elements.
